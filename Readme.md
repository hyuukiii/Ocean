### Windows OS에서는

# 메뉴 방식
ocean.bat

# cmd에서 직접 명령
ocean.bat start
ocean.bat stop
ocean.bat mysql

#### 배치 파일 사용법
- `start.bat` - 도커 시작
- `stop.bat` - 도커 중지
- `reset.bat` - 도커 초기화 (모든 데이터 삭제)
- `status.bat` - 컨테이너 상태 확인
- `mysql.bat` - MySQL CLI 접속
- `logs.bat` - 실시간 로그 확인
