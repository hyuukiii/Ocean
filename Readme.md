### Windows OS 에서 도커 사용 방법

# 실행 하는 방법
ocean 파일 에서 ocean.bat 실행 하기

# cmd에서 직접 명령
ocean.bat start  
ocean.bat stop  
ocean.bat mysql  

#### 배치 파일 사용법
- `start.bat` - 도커 시작
- `stop.bat` - 도커 중지
- `reset.bat` - 도커 초기화 (모든 데이터 삭제)
- `status.bat` - 컨테이너 상태 확인
- `mysql.bat` - MySQL CLI 접속
- `logs.bat` - 실시간 로그 확인
